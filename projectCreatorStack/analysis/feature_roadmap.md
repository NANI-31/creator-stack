# Feature Roadmap

## MVP Phase (In Progress)

- [x] Basic project structure and boilerplate.
- [ ] User authentication (Login/Signup).
- [ ] Tool listing CRUD (Create, Read, Update, Delete).
- [ ] Basic categorization and filtering.
- [ ] Voting system (Upvote/Downvote).

## Advanced Phase

- [ ] Reddit-style nested comments.
- [ ] Complex ranking algorithm (score = (upvotes - downvotes) + (avg_rating \* 2)).
- [ ] Request system for new additions.
- [ ] User profiles with contribution history.
- [ ] Moderator dashboard for submission approval.

## Future Possibilities

- [ ] Personalized recommendations based on user interests.
- [ ] Integration with tool APIs for real-time stats (e.g., GitHub stars for dev tools).
- [ ] Chrome/Firefox extension for quick tool submission.
